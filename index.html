m-mode unitary decomposition

Decompose an n×n unitary U using the m-mode (RQ / Tetris) procedure. The page shows embedded Q̃k† per step as tables; raw JSON is available via download. Final D is shown below. Runs locally in your browser (Pyodide + NumPy). Serve this folder over HTTP (see below); file:// may block loading.

Parameters
Unitary matrix U (JSON)

Format: each complex entry as [re, im], rows as JSON arrays. Example 2×2 identity: [[[1,0],[0,0]],[[0,0],[1,0]]] — or real-only [[1,0],[0,1]].

      m-mode circuit (schematic)
      

        
        
        
      

      

        Horizontal lines are modes (1…n); each column is one gate (step k =
        Q̃k†). Boxes span the active
        cols; the label is the RQ block size m′×m′ with
        m′ = len(cols). Wide or tall diagrams scroll inside the frame; hide the
        diagram above if the page feels too heavy.
      

      

        

      

    


    

      
        Steps — embedded
        Q̃k†
        (n×n)
      
      

        Each step lists active column indices cols. Step k uses
        \tilde{Q}_{k}^{\dagger} (embedded n×n). Entries are
        a ± bi. Use the download buttons for JSON.
